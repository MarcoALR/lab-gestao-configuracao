# Laboratório de Gestão de Configuração Moderna com Git e GitHub

## Descrição

Este repositório foi desenvolvido como parte da disciplina de Engenharia de Software, na atividade prática de Gestão de Configuração Moderna com Git e GitHub.

O objetivo deste projeto é aplicar conceitos fundamentais de controle de versão utilizando Git e GitHub, incluindo:

* Criação e gerenciamento de repositórios;
* Versionamento de código-fonte;
* Uso de branches para desenvolvimento de funcionalidades;
* Realização de commits seguindo o padrão Conventional Commits;
* Integração de alterações através de merges;
* Publicação e sincronização de código em repositórios remotos.

## Funcionalidades

* Programa "Hello World";
* Mensagem de despedida ("Adeus!");
* Histórico de commits utilizando boas práticas de versionamento;
* Demonstração de fluxo de trabalho com branches e merges.
